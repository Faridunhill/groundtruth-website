## TARGET LIST — Who to contact

### Priority 1: Nashville / Brentwood / Franklin

Search on Zillow Agent Finder and Realtor.com:
- Filter: Buyer's agents in Williamson County, Davidson County
- Sort by: Reviews / Recent sales
- Target: Agents with 10+ transactions/year, out-of-state buyer mentions in bio

### Priority 2: Knoxville / Knox County

- Same search method
- Focus: Agents near Oak Ridge, Maryville — high corporate relocation activity

### Priority 3: Mt. Juliet / Wilson County

- Our home county — local credibility is highest here
- Target: Agents who list in the $300k–$600k range (our buyer's sweet spot)

### Where to find email addresses

1. Zillow Agent profile → usually lists email or website
2. Realtor.com Agent directory
3. Agent's personal website (Google "[Agent Name] Nashville realtor")
4. Tennessee Real Estate Commission license lookup: tre.tn.gov → public license search
